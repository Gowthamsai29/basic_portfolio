NISITA App

QR Code-Based Security Solution for Educational Institutions

Abstract

The rapid advancements in technology have opened new avenues for enhancing security and efficiency in educational institutions. This project focuses on developing a mobile application for NIT Warangal that leverages QR code scanning to streamline the process of tracking student activities. Traditional methods such as manual attendance logs and biometric systems present several limitations, including inefficiency, susceptibility to errors, and ineffectiveness during pandemics. The proposed system addresses these issues by providing a secure, reliable, and user-friendly solution for security officers.

Introduction

In today's fast-paced world, ensuring the security and efficient management of student movements within educational institutions is paramount. National Institute of Technology Warangal (NITW) is no exception, with its large campus and substantial student body necessitating robust and streamlined security protocols. Traditional methods of monitoring student entry and exit, such as manual logbooks, are not only time-consuming but also prone to errors and inefficiencies.

Problem Statement

The current system for tracking student entry and exit at NITW involves manual logging and biometric systems. While biometrics offer certain advantages, they also present significant challenges. The biometric machines often overheat, fail to capture impressions accurately, and are not viable during health crises such as the COVID-19 pandemic. Manual logging, on the other hand, is inefficient, prone to human error, and lacks real-time data accessibility. There is a clear need for a more accurate, efficient, and reliable system to monitor student movements. Our project aims to address these issues by leveraging QR code technology to automate the tracking process, ensuring precise and timely records of student entries and exits.

Literature about Similar Works

Several institutions have implemented automated attendance and tracking systems using various technologies such as RFID, biometric scanning, and mobile applications. RFID systems, while effective, can be costly and require substantial infrastructure changes. Biometric systems offer high accuracy but are susceptible to issues such as overheating, failed impressions, and are unsuitable during pandemics due to hygiene concerns. Mobile applications using QR codes have emerged as a cost-effective and efficient alternative, offering ease of use and minimal infrastructure requirements. These applications ensure contactless operation, which is crucial during health crises, and provide real-time data for improved monitoring and security.

Proposed System

The proposed system is a mobile application that uses QR code scanning to track student activities. The application is designed to provide a user-friendly interface for security officers to log in, scan QR codes, and manage scanned data. The main features include user authentication via phone number and OTP, QR code scanning, timestamping of scanned data, and options to mark and share suspicious entries.

Objectives

  • Enhance Security: Provide a reliable system for security officers to track and verify student activities.
  • Improve Efficiency: Streamline the process of activity tracking, reducing manual errors and saving time.
  • User-Friendly Interface: Ensure the application is easy to use for security officers with minimal technical training.
  • Data Management: Facilitate the efficient management, categorization, and sharing of scanned data.

System Components

  • User Authentication:
    • Firebase Authentication: Utilizes Firebase for secure login using phone numbers. Upon entering their phone number, security officers receive a 6-digit OTP (One Time Password) via SMS for verification.
    • ReCAPTCHA Verification: Ensures that the login process is secure and prevents automated bot logins.
  • User Interface: Intuitive design with launcher page for phone number entry and main interface post-login.
  • QR Code Scanning:
    • Camera Permission: Requests permission to access the device's camera.
    • Scanning Functionality: Captures QR code and timestamps scanned data.
  • Data Management:
    • Pop-up Dialog: Displays student details upon successful QR code scanning, with options to share, copy text, or mark as favorite.
    • Recent Scanned & Favorites: Sections that list recently scanned QR codes and entries marked as favorite.

Detailed Workflow

Login Activity:

  • Phone Number Entry: Security officers enter their phone number on the launcher page.
  • OTP Verification: Firebase sends a 6-digit OTP to the entered phone number. The officer enters the OTP to gain access to the main interface.
  • ReCAPTCHA Verification: Ensures that the login process is secure.

QR Code Scanning:

  • Initiate Scanner: The officer taps the "Scan QR" button, prompting the app to request camera permission.
  • Permission Handling: If the officer denies permission, the app will request it two more times. If permission is still denied, the app crashes.
  • Scanning Process: Once permission is granted, the scanner activates, capturing the QR code and timestamping the scan.

Data Management:

  • Categorization: Scanned data is categorized into "Recent Scanned" and "Favorites" sections.
  • Sharing Options: Allows the security officer to share scanned details via WhatsApp, text message, or other available apps.

Design Details

Front-end:

  • User Interface Design: Intuitive and user-friendly interface with clear navigation and controls.
  • Error Handling: Ensures that the app handles errors gracefully, such as repeated permission requests and app crashes.

Back-end:

  • Firebase Integration: Utilizes Firebase for user authentication and room database management.
  • Data Storage: Efficiently stores and retrieves scanned data, ensuring quick access and minimal latency.

Conclusion

The mobile application developed for NIT Warangal effectively addresses the limitations of traditional student tracking methods. By leveraging QR code technology and mobile interfaces, this app offers a more efficient, reliable, and user-friendly solution for security officers. It enhances security by enabling officers to quickly scan, manage, and share student data, thus improving overall operational efficiency.
